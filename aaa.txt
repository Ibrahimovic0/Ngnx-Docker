		Imagen Hello-World

docker run hello-world
Muestra que pasa para mostrar el hola mundo.

Para generar este mensaje, Docker siguió los siguientes pasos:
 1. El cliente de Docker se puso en contacto con el daemon de Docker.
 2. El daemon de Docker extrajo la imagen "hello-world" del Docker Hub.
    (amd64)
 3. El daemon de Docker creó un nuevo contenedor a partir de esa imagen que ejecuta el
    ejecutable que produce la salida que está leyendo actualmente.
 4. El daemon de Docker transmitió esa salida al cliente de Docker, que la envió
    a su terminal.
